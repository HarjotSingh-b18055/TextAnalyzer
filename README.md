# TextAnalyzer

Install tkinter:
```bash
sudo apt-get install python3-tk
```

Install ttkthemes:
```bash
python3 -m pip install git+https://github.com/RedFantom/ttkthemes
```

Install matplotlib:
```bash
python3 -m pip install -U matplotlib
```
Install nltk
```bash
pip install nltk
```

Run using:
```bash
python3 TextAnalyzer.py
```
